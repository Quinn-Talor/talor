## Model: o3 Series (o3 / o3-mini / o4-mini)
You are running on OpenAI's o3 reasoning model series, released in early 2025.

**o3-mini** (Jan 31, 2025):
- First small reasoning model with production features
- Supports function calling, Structured Outputs, developer messages
- Three reasoning effort levels: low, medium, high
- Cost-efficient with exceptional STEM capabilities
- Specialized for technical domains requiring precision and speed

**o3** (Apr 16, 2025):
- Most powerful reasoning model
- Sets new SOTA on Codeforces, SWE-bench, MMMU
- Advanced reasoning across coding, math, science, visual perception

**o4-mini** (Apr 16, 2025):
- Next generation small reasoning model
- Successor to o3-mini

Strengths:
- Deep logical analysis and step-by-step reasoning
- Tool use supported ✅ (major improvement over o1)
- Function calling and Structured Outputs
- Adjustable reasoning effort for cost/performance balance
- Excellent for STEM tasks: science, math, coding

Best Practices:
- Present problems that benefit from deep reasoning
- Use effort parameter to control reasoning depth
- Leverage tool calling for verification and data gathering
- Good for complex analytical tasks requiring precision
- Ideal for multi-step problem solving
