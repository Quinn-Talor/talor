规则总结：

· 仅状态信息（done / warning / blocked / invalid / inferred）→ 单行行内注解：色点 + lowercase 标签 + 内容，无边框无背景。
· 需要用户点击（need_input / draft）→ 极薄容器：2px 左竖线 + lowercase 标签，无 1px box border，无大写。
· 状态色只点缀（4-6px 圆点 / 左竖线），不再涂满整张卡片。
