简而言之，ReactLoop 是 Talor 主进程里的核心控制流。

核心步骤

  1. 从 PromptPipeline 拉到完整 messages
  2. 调用 provider 流式 generate
  3. 每个 tool_use 进 registry 执行
  4. 结果回灌 → 下一轮，直到 done 或上限

不变量

assistant(tool_use) 和 tool(result) 必须同事务落盘 — 一旦破坏配对，session 永久废。详见 standards.md §I-MUST-1。

关键参数

参数 默认 作用
maxIterations 20 死循环上限
contextWindow 200k 触发压缩阈值
stepIdCheck on halt 检测开关
