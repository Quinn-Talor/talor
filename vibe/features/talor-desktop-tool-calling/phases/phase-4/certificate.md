## Demo 验证（必须亲自运行，不允许假设）

### Demo 场景

**操作步骤**：
1. 启动 talor-desktop 应用（`cd talor-desktop && npm run dev`）
2. 连接一个 LLM Provider（确保 AI 可用）
3. 在 AI 对话中输入：「请帮我运行 `ls -la` 命令」
4. 继续输入：「请帮我运行一个耗时命令，比如 `sleep 5`」
5. 继续输入：「请帮我运行 `pwd` 命令」
6. 连续发送多个工具调用请求，验证多轮工具调用

**预期可观察结果**：
> 用户在 AI 对话中发送命令请求 → AI 调用 bash 工具执行 → AI 响应命令输出 → 用户看到命令执行结果
> 超时命令 → AI 响应超时错误
> 多轮调用 → AI 依次执行并返回结果
