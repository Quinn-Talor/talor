**状态说明**：

## 使用规则

1. **只记录，不实现**：添加到此表后，继续当前会话目标
2. **每次会话结束时**：通知人类此文件有更新，请确认 pending 项
3. **人类确认后**：将 approved 的功能加入 `.claude.md §3.2` 对应阶段
4. **不允许**：不记录直接实现，或因为"功能太小"就跳过记录
