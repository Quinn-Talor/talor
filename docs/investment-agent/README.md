## 0. 第一性原理:散户在 A 股的位置

**市场结构**:A 股 = 人民币计价的境内股票,分三所(沪/深/北)四板(主板/创业板/科创板/北交所),代码开头即风险等级(60/00 主板 ±10%;30/688 ±20%;8xx ±30%)。规则硬约束:T+1、涨跌停、最小 1 手。

**散户的位置(不对称)**:占账户数 99%+、贡献大部分成交量,但持流通市值仅约 1/4。结构性劣势在 **信息、资金、纪律** 三方面。

**散户唯一能赢的三个非对称优势**(机构做不到的):

1. **时间维度** —— 没有考核压力,能拿 3~5 年。
2. **资金灵活** —— 船小好掉头,且**可以长期空仓等机会**(机构必须满仓)。
3. **逆向能力** —— 能在群体恐慌/贪婪时反向。

**认知重塑三条铁律**:

- 在 A 股,"勤奋"常是负资产,**"克制"才是正资产**。
- 你的优势不是"更快更准",而是"**更慢、更稳、更逆向**"。
- 最高级的认知是**诚实评估自己**:走指数定投(A 路线)还是建个股体系(B 路线)。本文档服务 **B 路线**。

## 2. 三层分析体系:宏观 → 行业 → 标的

A 股是全球少数 **宏观/政策权重 > 基本面权重** 的市场("政策市 + 流动性市"),所以必须自上而下加宏观层。
**三层是"约束关系"**:上层为下层划边界(↓),下层为上层做验证(↑)。

```
宏观层(环境底座, agent 主导, 季度级)→ 产出: ①仓位基调 ②风格基调 ③赛道过滤名单
   ↓ 约束
行业层(选景气赛道, 人机协作, 月/周)→ 产出: 1~3 个景气向上行业
   ↓ 约束
标的层(逻辑卡 7 要素, 人主导, 季报/事件)→ 产出: 逻辑卡 v2 + 证伪追踪
```

> **宏观层的镣铐**:只做"环境识别"(流动性/周期/政策),**不做"点位预测/短线择时"**。它是地形图,不是天气预报。

## 3. 数据 → 逻辑 管道

数据是整个体系的地基:**逻辑不是拍脑袋,是从数据里逼出来的。** 本节把每一层、每一条逻辑反向拆到它依赖的原始数据 + 数据逻辑。完整数据流见图 [`data-framework.svg`](./data-framework.svg)。

### 3.0 总范式:五步加工链 + 预期差

任何结论都走同一条加工链:

```
①原始数据 → ②衍生指标(同比/环比·占比·历史分位·趋势·财务勾稽)
          → ③三维对比(历史自比·同业横比·vs市场预期 ★)
          → ④信号(扩张/收缩·便宜/贵·超预期/不及·排雷红灯)
          → ⑤逻辑产出(仓位基调 / 赛道过滤 / 逻辑卡某要素)
```

> 🔑 **核心洞察:超额收益 = 数据揭示的「预期差」,而非数据绝对值。**
> 实际数据 vs 市场一致预期(consensus)。利好若已被预期,出来反而跌(price in)。一家公司利润增 30%,若市场预期 40%,数据一出就跌。
> 终点不是"算出真相",而是"**找到市场还没定价的真相**"。

> 取数纪律(贯穿全节):**对齐 as-of 日期 + 交叉验证 + 自己重算**,不盲信单一源的 published 倍数(同一标的常从不同源拿到不同"当前价"/PE)。

### 3.1 宏观层数据

**产出① 仓位基调(进攻/防守)← 流动性 + 风险偏好**

| 原始数据                     | 来源            | 频率  | 数据逻辑                                                     |
| ---------------------------- | --------------- | ----- | ------------------------------------------------------------ |
| M2、社融增速、社融存量同比   | 央行/统计局     | 月    | 社融增速**拐点**领先;M2−社融剪刀差看资金空转                 |
| DR007、LPR、10Y 国债收益率   | 央行/中国货币网 | 日    | DR007 vs 政策利率→松紧;**10Y↓→无风险利率↓→抬估值**(DCF 分母) |
| 央行公开市场操作(逆回购/MLF) | 央行            | 日    | 净投放→放水信号                                              |
| 北向资金净流入               | 交易所          | 日    | 持续流入→外资 risk-on                                        |
| 两融余额、成交额/换手率      | 交易所          | 日    | 两融↑+放量→杠杆+情绪升温                                     |
| 新成立基金份额               | 基金业协会      | 周/月 | 爆款发行→情绪顶(常**反向**)                                  |

> 合成:宽松(社融↑/利率↓/净投放)+ 偏好高(北向流入/两融升/放量)→ **进攻**;反之防守。新基金发行常是反向指标。

**产出② 风格基调(价值/成长、大/小盘)← 经济周期 + 利率**

| 原始数据                         | 来源        | 频率 | 数据逻辑                                          |
| -------------------------------- | ----------- | ---- | ------------------------------------------------- |
| PMI(官方+财新)、工业增加值、社零 | 统计局/财新 | 月   | PMI>50 扩张;财新偏中小/出口                       |
| CPI、PPI                         | 统计局      | 月   | **PPI−CPI 剪刀差**→上下游利润分配;PPI 上行→周期股 |
| GDP                              | 统计局      | 季   | 增长方向定美林时钟象限                            |
| 10Y 国债收益率                   | 中国货币网  | 日   | **利率↑→价值占优**(成长股久期长);↓→成长占优       |

> 合成:美林时钟 = f(增长↑↓, 通胀↑↓)。复苏→成长;过热→周期/资源;滞胀→现金/防御;衰退→债券/高股息。

**产出③ 赛道过滤名单 ← 政策(A 股最特殊)**

| 原始数据                                   | 来源          | 频率     | 数据逻辑                                                                                            |
| ------------------------------------------ | ------------- | -------- | --------------------------------------------------------------------------------------------------- |
| 五年规划、政府工作报告、中央经济工作会议   | 政府网/新华社 | 年/季    | 长期国家战略方向                                                                                    |
| 部委专项政策、产业目录、补贴/集采/限产文件 | 各部委        | 事件驱动 | **文本解析**:鼓励词(支持/培育/补贴/国产化)vs 限制词(规范/整顿/压减/反垄断)→ 顺风/中性/逆风/一票否决 |

> 这是**文本→标签**,核心是"避雷"(教培、地产、医药集采都是先例)。

### 3.2 行业层数据

| 维度       | 原始数据                              | 来源                               | 数据逻辑                                     |
| ---------- | ------------------------------------- | ---------------------------------- | -------------------------------------------- |
| 景气(量价) | 产销量、产品价格(猪价/面板/水泥/装机) | 行业协会、统计局分行业、大宗数据商 | **量价齐升→景气上行**;价格是**高频先行**指标 |
| 库存周期   | 行业库存、产成品存货同比              | 统计局/Wind                        | **低库存+需求回升=主动补库存(景气拐点)**     |
| 产能       | 产能利用率、在建工程、资本开支        | 统计局、上市公司汇总               | 过剩→价格战;出清→盈利修复                    |
| 估值分位   | 行业 PE/PB 历史分位、行业 ROE         | 交易所/三方                        | **低分位+景气上行=戴维斯双击候选**           |
| 资金流向   | 行业北向持仓、ETF 份额、板块成交占比  | 交易所/ETF 披露                    | 持续流入→机构在配置                          |

> 合成:景气 + 估值便宜 + 资金流入 三共振;背离时以**景气和估值**为准,资金面只作佐证。

### 3.3 标的层数据(逐逻辑卡要素映射)

| 逻辑卡要素         | 需要的数据                                       | 数据逻辑                                  |
| ------------------ | ------------------------------------------------ | ----------------------------------------- |
| ②收益来源-成长     | 营收、**扣非**净利润增速(连续多季)               | 看扣非才是真成长;连续性比单季重要         |
| ②收益来源-估值修复 | PE/PB 历史分位、PEG                              | PEG<1 且低分位→修复空间                   |
| ②收益来源-分红     | 股息率、分红率、回购金额                         | 高且稳→现金牛                             |
| ③公司α-护城河      | 毛利率 vs 同业、ROE 杜邦拆解、销售费用率         | **毛利率持续高于同业且稳定→护城河**       |
| ③公司α-份额        | 市占率(营收/行业规模)趋势                        | 份额提升=α 验证                           |
| ③催化剂            | 在建工程、合同负债/预收款、新签订单、资本开支    | **在建工程↑→未来产能;合同负债↑→未来收入** |
| ④关键假设验证      | 对应假设的行业销量/毛利率序列/研发投入           | 逐条假设挂一个可监控数据                  |
| ⑥赔率              | 估值输入(自由现金流、增速、折现率)、vs 一致预期  | 见估值五步                                |
| ⑦反方              | 融券余额、限售解禁时间表、机构评级下调、北向减持 | 解禁前抛压;融券升=有人做空                |

**财务排雷的数据逻辑(每条是一个勾稽/对比)**:

| 红灯信号                          | 数据逻辑(为什么是雷)          |
| --------------------------------- | ----------------------------- |
| 经营现金流/净利润 < 1(持续)       | 赚的是"白条"不是真钱          |
| 应收账款增速 >> 营收增速          | 放宽信用 / 收不回 / 虚增      |
| 存货增速 >> 营收增速              | 滞销、积压、减值前兆          |
| 商誉/净资产过高                   | 商誉减值雷                    |
| 有息负债/EBITDA 高、利息保障低    | 偿债危机                      |
| 大股东高质押 / 密集减持           | 治理与爆仓风险                |
| 审计意见非标                      | **一票否决**                  |
| 存贷双高(财务费用<0 但货币资金高) | 货币资金造假嫌疑(康得新/康美) |

> **核心:三表勾稽** —— 利润表说赚钱→现金流量表验"钱到账没"→资产负债表验"是否堆在应收/存货"。**对不上,信现金流。** 这是对抗 A 股造假的第一道防线。

## 4. 企业逻辑三维体检

把"公司 α"拆成三个可评级维度。比喻:**业务=赛道和车 / 财务=仪表盘 / 管理=司机**。

| 维度            | 核心问题              | 关键看点                                                                  | 红旗                                                     |
| --------------- | --------------------- | ------------------------------------------------------------------------- | -------------------------------------------------------- |
| **业务逻辑**    | 能不能赢?             | 商业模式、护城河、份额趋势、收入质量                                      | 模式说不清、护城河靠单一关系、靠框架订单、客户集中       |
| **财务逻辑**    | 赢了没、真钱吗?       | 盈利(ROE杜邦/扣非)、现金(经现/净利≥1)、安全(应收存货/商誉/负债)、成长质量 | 利润高现金差、应收存货暴增、商誉高、连亏、减值、存贷双高 |
| **管理逻辑** ⚠️ | 人坑不坑你?(一票否决) | 承诺兑现度、诚信信披、增减持/质押、有无实控人、关键人                     | 跨界蹭概念、高管离职、高质押+减持、被问询、画饼不兑现    |

**优先级**:业务(前提)→ 财务(验证)→ 管理(底线/一票否决)。
铁律:**三表对不上信现金流;看管理行动不看言论。**

## 5. 逻辑卡 v2(决策载体)

标的层的最终产出物,每只票一张。下方九节骨架是**方法论原型**;落地的系统设计见 [`mvp-design.md`](./mvp-design.md),其中已把它重构为**事实 / 支点 / 判断三区 + 不可变快照时间线**(支点 = 承重的事实,证伪 = 支点变化)。九节骨架:

```
§0 标的定位(承接宏观/行业 + 估值快照, 对齐 as-of)
§1 核心命题(一句话, 可证伪 + 有时限)
§2 收益来源(成长/估值修复/分红, 主押哪个)
§3 企业逻辑三维体检 ★(业务/财务/管理 → 通过/存疑/否决)
§4 因果链(行业β + 公司α[来自§3] + 催化剂)
§5 关键假设(每条挂监控数据)
§6 证伪条件(§3红灯 + §5假设破坏 + 管理踩雷)
§7 估值与赔率(估值五步 + 预期差 + 赔率)
§8 反方观点(最强看空)
§9 决策与跟踪(决策规则 + 仓位 + 验证点 + 更新日志)
```

**内嵌决策规则**:

- ✅ 可买 = 三维通过 + 赔率为正 + 证伪未触发 + 安全边际足
- ⛔ 不碰/卖 = 管理否决 **或** 证伪触发 **或** 预期差为负且无安全边际

---

## 6. 估值五步

研究解决"好不好",估值解决"贵不贵"——**两件独立的事**。

1. **选乘数**(按行业/盈利:PE/PB/PS/PEG/EV-EBITDA)
2. **三维定位**(历史分位 + 同业横比 + vs 一致预期 → 便宜/合理/贵)
3. **逆向 DCF 现实性检验**:反推当前价隐含增速 → 合理还是离谱?(核心)
4. **安全边际**:估值下沿再打折买入
5. **卖出锚**:高估到某程度减仓

要点:估值是**区间不是点**、是**赔率工具不是水晶球**;PE 陷阱(周期股低 PE 在顶);好公司+合理价 > 平庸公司+便宜价(避价值陷阱)。

---

## 7. 交易纪律

把"知道"变成"做到"——因为情绪必然发作,**规则要事先写死、机械执行**。

- **仓位管理**(最重要):单一上限、分批建仓、总仓位挂宏观基调、赔率定大小、现金也是仓位。
- **买入**:逻辑通过 + 有安全边际才买;分批、不追高;基于逻辑不基于价格。
- **卖出**(只有三个理由):① 逻辑破坏(证伪触发,无条件卖)② 估值过高(卖出锚)③ 有更好机会。**价格跌不构成卖出理由。** 反处置效应:截断亏损、让利润奔跑。
- **止损**:逻辑止损(首选)> 价格止损(兜底);A 股波动大,纯价格止损易被洗出。
- **机械化**:交易计划前置、不盘中决策、降低看盘频率。

---

## 8. 复盘机制

让经验复利、体系自我进化。

- **复盘决策,不复盘结果**:好决策可能坏结果(运气)。四象限里重点复盘"决策错+结果好(侥幸)"和"决策错+结果坏(教训)"。
- **载体 = 逻辑卡回看**:当初假设对吗?证伪触发时执行了吗?反方说中了吗?(写入 §9 日志)
- **维度**:研究 / 估值 / **执行(知行差距)** / **归因(运气 β vs 能力 α)**。A 股情绪市,归因尤其重要——别把运气当能力。
- **产出**:个人「错误清单」+「原则库」+ 能力圈边界更新 → 反哺研究。

## 9. 两个案例(判别力验证)

同处 AI 算力赛道、同样大跌,框架给出相反结论:

| 维度      | 中际旭创 300308                           | 鸿博股份 002229                                    |
| --------- | ----------------------------------------- | -------------------------------------------------- |
| 业务      | ✅ 全球份额领先、毛利率升至 46%、量价齐升 | ❌ 印刷主业、英伟达关系存疑、靠框架订单            |
| 财务      | ✅ 营收利润高增、业绩翻倍兑现             | ❌ 连亏 6 年、2024 巨亏 2.9 亿、减值、预付却订单黄 |
| 管理      | ✅ 战略专注、持续兑现                     | ❌ 蹭概念、关键人解聘反目、画饼、深交所问询        |
| 估值/赔率 | (2025/5)恐慌错杀,预期差**正**             | 概念透支未兑现故事,预期差**负**                    |
| 证伪条件  | 未触发(北美 CSP capex 上修)               | **全部触发**                                       |
| **决策**  | **逆向买入**                              | **回避(价值陷阱)**                                 |

**最深一课 —— 两种"跌"本质不同**:中际旭创的跌是「市场错了」(情绪),鸿博的跌是「市场对了」(基本面证伪)。**三维体检 + 证伪条件,正是区分这两种跌的工具。**

> 关键对照:同一现象"预付款暴增",中际旭创后续兑现成收入(对未来有信心),鸿博却订单解除(钱出去收入没回)——**数据不能孤立看,要看因果链是否闭环。**
>
> 数据质量教训:同一标的从三个源拿到三个"当前价"——**取数必须对齐 as-of、交叉验证、自己重算。**

## 10. 对 Agent 的设计启示

**核心定位**:agent 不该帮散户"赌得更猛",而该**当外置的纪律和记忆**——补散户最致命的"扛不住情绪"和"记不住教训"两块短板。

**人机边界**:

| 体系环节 | agent 能做(确定性/数据密集,主场)                                 | 必须留给人         |
| -------- | ---------------------------------------------------------------- | ------------------ |
| 宏观层   | 仪表盘聚合、政策监控打标、周期定位、仓位/风格基调建议            | 拍板               |
| 行业层   | 景气/估值分位/资金流向取数                                       | 景气判断           |
| 业务逻辑 | 供料(毛利率、份额、行业数据)                                     | **护城河判断**     |
| 财务逻辑 | ✅ 全自动:勾稽、排雷、四象限                                     | 设阈值             |
| 管理逻辑 | 抓减持/质押/问询/高管变动、**承诺-兑现追踪**                     | **诚信判断**       |
| 估值     | ✅ 乘数/分位/同业/逆向DCF/敏感性                                 | 给假设、判断合理性 |
| 交易纪律 | **外置纪律**:证伪/卖出锚/仓位**预警**,拦截追涨杀跌               | 决策买卖           |
| 复盘     | **记录+归因**:存档逻辑卡、平仓对照、运气vs能力归因、维护错误清单 | 反思               |

**候选功能模块(待 B 阶段细化)**:

1. 数据采集与对齐层(Tushare/AkShare/cninfo/官网爬虫 → SQLite,对齐 as-of + 重算)
2. 宏观环境仪表盘 + 政策监控
3. 行业景气榜
4. 逻辑卡引擎(结构化 §0–§9,强制填证伪条件)
5. 财务排雷器(确定性规则,走代码)
6. 估值计算器(含逆向 DCF + 敏感性)
7. 证伪追踪器(季报/事件自动比对关键假设,触发预警)
8. 纪律预警器(卖出锚 / 仓位 / 反追涨杀跌)
9. 复盘归因器(逻辑卡对照 + 错误清单)

> 工程纪律(承接 CLAUDE.md §4.5):**"必须"级硬规则走代码**(排雷、勾稽、分位、证伪比对);LLM 只做政策文本解析、定性辅助、提问清单等软任务,不替人下定性结论。
